Build the backend/admin system for the Grays Park Masjid website frontend.

The backend must allow masjid staff/admins to manage the website without needing a developer. This is not just a basic CMS. It should be a Masjid Management Website System covering prayer times, announcements, events, courses, donations, enquiries, media, services, team members, and website settings.

Build a secure admin panel with login and role-based access.

Admin roles:

1. Super Admin

* Full access to everything

2. Masjid Admin

* Manage content, prayer times, announcements, events, classes, donations, enquiries, services, gallery, news, settings

3. Education Admin

* Manage courses, classes, teachers, student registrations

4. Donation Admin

* Manage donation campaigns and donation page content

5. Content Editor

* Manage news, gallery, announcements only

6. Read Only

* View dashboard and enquiries but cannot edit

Admin dashboard must show:

* Today’s prayer times
* Next prayer
* Active announcements
* Upcoming events
* New enquiries
* Open class registrations
* Active donation campaigns
* Recent form submissions
* Quick action buttons:
    * Add Announcement
    * Update Prayer Times
    * Add Event
    * Add Course
    * Add Donation Campaign
